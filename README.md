# 🚀 Developer Burnout Analysis

A data analytics project analyzing burnout patterns among **7,000 software developers** using **Python, SQL, and Power BI**.

---

## 📌 Project Overview
This project investigates how **workload, lifestyle, and behavioral factors** contribute to developer burnout. It identifies high-risk groups and delivers actionable insights to improve productivity and wellbeing.

---

## 🛠️ Tools & Technologies
- **Python** (Pandas, NumPy, Matplotlib, Seaborn)
- **SQL** (MySQL)
- **Power BI** (Dashboard & DAX)
- **Jupyter Notebook**

## 🔍 Key Insights
- **75%** of developers fall into Medium or High burnout
- **Stress level** is the strongest predictor of burnout
- **Long work hours + low sleep** significantly increase burnout risk
- Developers aged **20–25** are the most affected group

---

## 📊 Dashboard Features
The Power BI dashboard includes:
- Executive Overview

- Burnout Deep Dive

---

## ⚠️ Critical Finding
Developers working:
- **9+ hours/day**
- **<5 hours sleep**
- **High caffeine intake**

→ Have a **60% probability of High Burnout**

---

## 💡 Recommendations
- Cap daily work hours at **8 hours**
- Reduce unnecessary meetings
- Encourage better **sleep and exercise habits**
- Implement **stress monitoring systems**

---

## 🚀 Conclusion
Burnout is primarily driven by **stress and overwork**. Organizations must actively invest in developer wellbeing to sustain productivity and retention.
